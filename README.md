4004
============================================================
Intel 4004 emulator

Requirements
--------------------------------------------------

* Go (>= 1.10)

4004 is written by Go only with standard packages.
So, you can build 4004 by:
```
$ go build
```

Examples are written as [XXX](https://github.com/ujtakk/xxx) programs.
